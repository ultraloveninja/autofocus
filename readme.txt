== Installation ==

In your admin panel, go to Appearance > Themes and click the Add New button.
Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
Click Activate to use your new theme right away.
== Frequently Asked Questions ==

= Does this theme support any plugins? =

AutoFocus supports all standard WordPress plugins. It has enhanced support for Jetpack's portfolio features.

= How do I set up the homepage grid layout? =

The homepage grid layout automatically displays your latest posts with featured images. Simply ensure that each post has a featured image set.

= How do I display EXIF data for my photos? =

When you upload photos to WordPress, the EXIF data is automatically extracted if it exists in the original file. When viewing an image attachment page, the EXIF data will be displayed below the image.

= Can I customize the colors and fonts? =

Yes, you can customize basic colors through the WordPress Customizer. For more advanced customizations, you may want to create a child theme.
